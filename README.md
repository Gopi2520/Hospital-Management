#  Hospital Management System

## 📌 Project Overview

The **Hospital Management System** is a Java-based application designed to manage hospital operations efficiently. It allows users to handle patient records, doctor details, and hospital information using a structured database system.

This project demonstrates core concepts of **Java, JDBC, and database management**, making it suitable for learning full-stack backend development fundamentals.

---

## Features

### 👤 Patient Management

* Add new patient details
* View patient information
* Update patient records
* Delete or discharge patients
* Track patient status (Admitted / Discharged)

---

### 👨‍⚕️ Doctor Management

* Add doctor details
* View doctor information
* Update doctor details
* Delete doctor records
* View doctors based on specialization

---

### 🏢 Hospital Management

* Add hospital details
* View hospital information
* Update hospital records
* Delete hospital data

---

## 🛠️ Technologies Used

* **Java (Core Java)**
* **JDBC (Java Database Connectivity)**
* **MySQL Database**
* **SQL (CRUD Operations)**

---

## 🗄️ Database Structure

### Tables Used:

* `PATIENT`
* `DOCTOR`
* `HOSPITAL`

### Key Relationships:

* Each patient is assigned to a doctor
* Each doctor is associated with a hospital

## ⚙️ How to Run the Project

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Open the project in IDE (Eclipse / IntelliJ)

3. Configure database:

* Create MySQL database
* Update DB credentials in your code

4. Run the main Java file
